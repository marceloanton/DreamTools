# DreamTools

Private co-op utility mod for Shape of Dreams.

## Download

Use `DreamTools-v0.4.1.zip` from this repository.

## Install

1. Close Shape of Dreams.
2. Extract `DreamTools-v0.4.1.zip`.
3. Copy the extracted `DreamTools` folder into:

```text
C:\Program Files (x86)\Steam\steamapps\common\Shape of Dreams\Mods
```

Final path should look like:

```text
C:\Program Files (x86)\Steam\steamapps\common\Shape of Dreams\Mods\DreamTools\about\metadata.json
```

4. Start the game and enable DreamTools from the mod manager.
